本页面提供成绩提醒设置详细说明。
当你的成绩在教务处更新时，系统将会给你的QQ/微信发送一条通知。


1.首先进入订阅管理界面



2.打开成绩订阅开关

点击“新活动通知（期末成绩提醒）”前面的开关，弹出提示，点击确定。



3.增加订阅次数

点击“新活动通知（期末成绩提醒）”这几个字，弹出订阅提示

勾选“新活动通知”，点击允许

成功后，“新活动通知（期末成绩提醒）”末尾的数字会增加，数字多大就提醒几科成绩。



4.进入成绩页面



5.打开成绩提醒开关

打开“订阅成绩更新提醒”前面的开关



6.收到成绩提醒

一分钟左右会收到一条成绩通知，内容为之前的某科考试信息，这表明系统正常运行，并且表明操作无误。
接下来，需要在订阅页面增加订阅次数，就可以一直收到提醒了。
